# 🎀 Hello Kitty Island Adventure

Landing page temática desenvolvida como parte da disciplina **Introdução à Tecnologia Web**, do curso de **Sistemas de Informação**.

O projeto explora conceitos fundamentais de **estruturação de páginas web** e **design visual**, utilizando HTML, CSS e JavaScript.
